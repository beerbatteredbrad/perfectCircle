# Neon Circle

A cyberpunk-themed precision drawing game where you test your ability to draw a perfect circle.

## Features
- **Visuals:** High-contrast neon aesthetic with glowing effects and particle systems.
- **Game Modes:**
  - **Free Draw:** Draw the best circle you can anywhere on the screen.
  - **Target Mode:** Trace a specific ghost circle to test your accuracy.
  - **Hard Mode:** Avoid the red danger zones while drawing!
- **Audio:** Synthesized sound effects for feedback and immersion.
- **Progression:** Local leaderboard tracks your top 5 scores.

## How to Play
1. Open `index.html` in your web browser.
2. Select your mode using the buttons at the top.
3. Click and drag (or touch and drag) to draw a circle.
4. Try to close the loop and match the perfect shape!
5. Aim for 95% or higher to trigger the "Perfect" celebration.

## Technologies
- HTML5 Canvas
- Web Audio API
- LocalStorage for high scores
